# VG_project
Code and data for the final report of the 10 week project course regarding prediction of VG scores, the natural gene dosage variation, using machine learning approaches.

In order to run "supervised_machine_learning_models.ipynb" you will need to extract the zip files "gb_randsearch_grid.zip" and "data.zip" and transfer the "gb_randsearch_grid.csv" file into the "data" directory.
